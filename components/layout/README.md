# Layout Components

Shared layout components belong here.

Current repo status: Capital Radar is still primarily a static HTML / Node-rendered site. Layout components are therefore documented and staged here before being wired into production.

Planned components:

- `RadarShell` — application shell / top navigation / page wrapper
- `SectionPanel` — repeated panel container with section header and actions
- `DashboardGrid` — responsive grid utilities for radar pages

Rule: layout components should not contain investment logic. They should only organize screens.
