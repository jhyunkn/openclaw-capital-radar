# Radar Components

Capital Radar-specific visual intelligence components belong here.

These components should translate market data, ticker data, technical structure, valuation, liquidity, catalysts, and thesis logic into decision-support visuals.

Rules:

1. Every radar component must be driven by a clear data shape.
2. Mock JSON must be used before live data integration.
3. A component must answer a decision question, not merely display information.
4. Standalone HTML prototypes do not belong here; only extracted reusable components or component specs do.

- `TickerDecisionCard.md`
